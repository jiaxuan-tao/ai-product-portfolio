# Prompt Manager v1.1 产品需求文档

## 版本目标

在不改变 v1.0「场景 → 提示词 → 版本」主结构的前提下，提升提示词的复用效率、迭代安全性与离线可用性。

## 功能范围

### 1. 版本差异对比

- 在提示词详情页打开版本对比窗口。
- 可选择任意两个历史版本，默认比较上一版本与当前版本。
- 统一视图按行展示新增、删除和未变化内容。
- 对比只读，不修改当前草稿或版本历史。

### 2. 变量模板与填充预览

- 使用 `{{变量名}}` 定义变量，变量名支持字母、数字、下划线、点和短横线。
- 自动识别正文中的变量并生成填充表单。
- 未填写变量继续保留占位符，避免误以为空值完成渲染。
- 变量默认值仅保存在当前浏览器；复制渲染结果会记录最近使用时间。

### 3. 筛选、排序与最近使用

- 提供当前场景、收藏、最近使用和回收站四种资源视图。
- 支持按更新时间、最近使用、名称排序。
- 支持单标签筛选；全文搜索仍覆盖场景、标题、摘要、正文和标签。
- 打开或复制提示词时更新最近使用时间。

### 4. 回收站与撤销

- 删除场景或提示词改为移入回收站，版本历史仍保留。
- 删除完成后提供即时撤销入口。
- 回收站支持恢复和永久删除。
- 恢复场景时，仅恢复随该场景一起删除的提示词；此前单独删除的提示词保持在回收站。

### 5. PWA 与离线

- 缓存静态应用资源，首次访问后可离线打开。
- 提供 Web App Manifest，支持兼容浏览器安装为桌面应用。
- 保持纯静态部署与 IndexedDB 本地存储，不新增服务端或远程数据请求。

## 非目标

- 不做移动端适配。
- 不做账号、云同步、多人实时协作或远程 AI 调用。
- 不改变现有品牌名、主色系统和核心页面层级。

## 验收标准

- v1.0 IndexedDB 数据升级后完整保留。
- v1.0 JSON 备份可正常导入并补齐 v1.1 默认字段。
- 五项核心功能均可在桌面端完成主流程。
- TypeScript、单元测试和生产构建通过。
- Chrome/应用内浏览器完成核心流程验证，GitHub Pages 可访问。
