# v0.2.0｜提示词组织能力

## 目标

让本地提示词资料库支持更细的组织与查找方式，使用户能按照主题、使用频率和个人项目习惯沉淀提示词。

## 本次范围

- 为提示词增加标签，支持中文逗号或英文逗号分隔输入。
- 在提示词卡片中展示标签，并将标签加入搜索范围。
- 支持收藏提示词，并提供“收藏”筛选入口。
- 支持新增自定义分类。
- 支持重命名自定义分类，并同步更新分类内提示词。
- 删除自定义分类时，要求选择一个迁移目标；分类内提示词只迁移，不删除。
- 导出升级为 `schemaVersion: 2`，额外包含自定义分类、标签和收藏状态。
- 保持 `schemaVersion: 1` 备份文件的导入兼容性。

## 本地数据

提示词新增可选字段：

- `tags`：字符串数组。
- `isFavorite`：布尔值。

自定义分类保存在独立的本地设置记录中。导入 v2 备份时，会合并文件中的自定义分类及新增提示词所属分类；导入 v1 备份时，标签和收藏使用默认值。

## 交互规则

- “全部”与“收藏”为系统筛选项，不能用作自定义分类名称。
- 默认分类不能重命名或删除。
- 自定义分类不能与默认分类或现有分类重名。
- 删除自定义分类前必须选择迁移目标。
- 恢复默认模板会清除自定义提示词、标签、收藏和自定义分类。

## 验证重点

- 标签和收藏保存后刷新页面仍存在。
- “收藏”筛选只展示已收藏提示词。
- 自定义分类创建、重命名、删除与迁移后，提示词数量正确。
- v2 导出包含新字段，v1 导入仍可成功。
- 搜索、复制、保存、恢复默认模板与 JSON 合并导入保持可用。

## 未包含内容

- 云同步、登录、后端和 AI API。
- 最近使用排序、模板版本历史和第三方工具数据互通。
