# Changelog｜Vibe Coding 提示词资料库

## v0.1.2｜本地数据迁移

### Added

- Added JSON export with `schemaVersion: 1` and export timestamp metadata.
- Added JSON import that merges valid prompts into the current local library.
- Added duplicate-ID detection and invalid-record skipping during import.
- Added v0.1.2 iteration documentation and README instructions for backup and restore.

### Changed

- Clarified that import and export only operate on local browser data and do not upload prompts.

### Not Included

- No cloud sync.
- No account or login.
- No backend service.
- No AI API integration.
- No tags, favorites, or custom category management yet.

## v0.1.1｜中文本地化版本

### 更新说明

- Completed Chinese UI localization.
- Updated default prompt categories and templates.
- Improved usability for Chinese Vibe Coding workflows.

### 中文说明

- 完成页面 UI 中文化，包括标题、分类、按钮、表单标签和交互反馈。
- 更新 8 个默认提示词分类、模板标题、说明和正文中的英文说明。
- 提升中文 Vibe Coding 工作流下的使用体验。

## v0.1.0｜Initial Remix

### Added

- Added Vibe Coding Prompt Library as a lightweight local-first prompt manager.
- Added default categories for PRD, Codex, UI refinement, debugging, README writing, changelog writing, user flow, and product structure.
- Added reusable prompt templates for common Vibe Coding tasks.
- Added project documentation including PRD, user flow, product structure, prompts, changelog, and reference notes.

### Changed

- Reframed the original prompt manager experience around personal Vibe Coding workflows.
- Updated product naming, page copy, and default information architecture.
- Adjusted visual presentation to keep the tool simple and documentation-oriented.

### Not Included

- No login.
- No cloud sync.
- No AI API integration.
- No backend service.
- No team collaboration.

### Next

- Add more project-specific prompt templates.
- Explore import / export support.
- Improve prompt tagging and filtering.
