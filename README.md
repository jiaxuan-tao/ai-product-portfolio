# AI Product Portfolio 🚀

这是一个用于整理个人 AI 产品实践项目的作品集仓库。

本仓库主要用于记录后续通过 Vibe Coding 方式完成或推进的 AI 产品项目，包括产品定位、用户场景、核心功能、需求文档、交互流程、Prompt 设计、开发记录和迭代思考。

当前阶段先搭建基础结构，后续会随着具体项目推进逐步补充内容。

---

## Projects

- Vibe Coding 提示词资料库
  面向 Vibe Coding 工作流的本地化提示词管理工具，用于整理和复用 PRD、Codex 指令、UI 修改、Debug 和项目文档提示词。

---

## Repository Structure｜仓库结构 📂

- `projects/`
  用于存放具体 AI 产品项目。每个项目会单独建立一个文件夹。

- `templates/`
  用于存放项目文档模板，包括 README、PRD、用户流程、产品结构、Prompt 和迭代记录模板。

- `assets/`
  用于存放项目截图、流程图、原型图或其他展示素材。
